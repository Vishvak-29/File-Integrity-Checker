# File Integrity Checker

## Description
A Python-based File Integrity Checker that generates and verifies SHA-256 hashes to detect whether a file has been modified.

## Features
- Generate SHA-256 hash of a file
- Verify file integrity
- Detect file modifications
- Uses Python's built-in hashlib module

## Technologies Used
- Python 3
- hashlib

## How to Run

```bash
python fileintegritychecker.py
```
